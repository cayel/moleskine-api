# Moleskine API

## Description
This API is developped with [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy

### Zeit

#### How to deploy

This API is deployed to the cloud with Now.
The configuration file is ```now.json```
To deploy the application

```bash
# build
$ npm run build

# deploy
$ now
```
